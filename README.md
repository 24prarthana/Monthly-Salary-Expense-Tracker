# Monthly Salary & Expense Tracker

A simple web application that helps users manage their monthly salary and track daily expenses.

## Features

- Enter monthly salary
- Add daily expenses with description
- Select expense category
- Record expense date
- Automatically calculate total spent
- Display remaining balance from salary
- View expense history
- Delete expenses

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. User enters their monthly salary.
2. User adds daily expenses with:
   - Description
   - Category
   - Date
   - Amount
3. The system calculates:
   - Total amount spent
   - Remaining balance from the salary.
4. All expenses are displayed in the expense history list.
